# End-to-End Sales Analytics & BI Pipeline

## Overview
This project demonstrates a complete data analytics workflow, transforming raw sales data into actionable business insights.

## Tools & Technologies
- Excel
- SQL
- Python (Pandas, NumPy)
- Power BI (Power Query, DAX, Measures, Slicers)

## Analytics Workflow
1. Raw sales data ingestion
2. Data cleaning and validation using Python
3. Analytical querying using SQL
4. Data transformation using Power Query
5. KPI modeling and DAX measures
6. Interactive dashboard development in Power BI

## Key Features
- Structured sales performance analysis
- Calculated measures and KPIs
- Interactive slicers and filters

## Impact
Enabled end-to-end sales performance analysis and supported data-driven business decision-making.
